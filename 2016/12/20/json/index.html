json

2016-12-20

JSON是一种数据表示的格式，包括json对象和json数组。

json的基本语法格式

对象： { 属性：值 ， 属性：值 }

数组：[对象，对象，对象]
获得一个数组的大小 使用 length属性。

访问一个对象的属性：json对象.属性。。。。可以取值也可赋值。
访问一个数组中的元素的方式：json数组[索引]

JSON文本是一个符合JSON语法格式的字符串。因为服务器只能返回文本。

JSON文本转换成JSON对象使用eval函数。

在服务器端，在Java类中三个状态之间的转换。

1 json文本
2 json对象/数组
3 java对象/集合
